# Sistema-de-Gesti-n-Acad-mica-e-Intranet
Aplicación web Full-Stack monolítica diseñada para gestionar usuarios, roles y catálogos de cursos académicos. El proyecto implementa un sistema de intranet seguro con autenticación de usuarios, control de acceso basado en roles (RBAC), y persistencia de datos en una base de datos relacional.
